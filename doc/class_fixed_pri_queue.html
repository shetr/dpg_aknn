Detailed Description

template<typename T>
class FixedPriQueue< T >

Shared interface for priority queues of fixed size. Used for k-NN implementation. The implementations should behave like normal priority queue, with the difference that it should remove elements with lowest priority if the capacity exceeds specified maximum.
